About SpamShield
Deep learning powered email classification using Bidirectional LSTM.
What is SpamShield?

SpamShield is a full-stack web application that uses a Bidirectional LSTM (BiLSTM) deep learning model to classify emails as spam or legitimate (ham).

Unlike rule-based spam filters, the BiLSTM model learns patterns from real email data — understanding context, word order, and subtle linguistic cues.

Trained on 5,576 real SMS messages from the UCI SMS Spam Collection dataset, achieving over 97% F1-score.

FAQ
Why BiLSTM instead of BERT?
With 15000 training samples, BERT (110M parameters) would overfit severely. BiLSTM with ~2.4M parameters is the sweet spot — powerful enough to learn spam patterns, small enough to generalize.
How does it handle new spam tactics?
The model learns word-level patterns and context rather than specific phrases, so it generalizes to unseen spam. You can retrain with new data using python app.py --train.
Is email data stored?
Yes — predictions are stored locally in history.db on your machine. Nothing is sent to any external server.
How accurate is it?
The model achieves ~97.8% F1-score on the test set. Very short or ambiguous emails may score near 50% confidence — the model is honest about uncertainty.
